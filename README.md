# Task 3 – Tic Tac Toe Web Application

 About the Task
This task involves building a simple Tic-Tac-Toe web application where two players can play against each other. The application tracks player moves, checks for winning conditions, and displays the game result.

 Technologies Used
- HTML5
- CSS3
- JavaScript

 Features
- Two-player gameplay
- Win and draw detection
- Restart game option
- Interactive user interface
